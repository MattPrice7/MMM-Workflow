# Econimap Source of Truth

The canonical Econimap package lives in this directory:

`/Users/mattprice/Documents/Codex/econimap`

All package code, tests, Stan code, package documentation, and implementation
designs must be changed here first. The contents of `inst/scripts/`,
`full_package_scripts/`, `mmm_latest_scripts/`, and
`MMM_Workflow/econimap/` are distribution, compatibility, or mirror surfaces.
They are synchronized from this package and must not be treated as independent
implementations.

The core standalone files in `inst/scripts/` are regenerated by
`tools/generate_standalone_scripts.R`. Never patch those generated files
directly. `Rscript tools/generate_standalone_scripts.R --check` is the drift
gate used by the focused test suite.

Current architectural design documents live under `docs/`.
